# Checklist AdSense — Emprego São Luís

Use esta lista antes de solicitar aprovação no Google AdSense.

## Infraestrutura

- [ ] Site publicado com domínio próprio (`empregossaoluis.com.br`)
- [ ] SSL ativo (HTTPS)
- [ ] `NEXT_PUBLIC_SITE_URL` apontando para o domínio final
- [ ] Favicon configurado (`public/favicon.ico`)
- [ ] Logo configurada (`public/logo.png`, `public/logo-horizontal.png`)
- [ ] Open Graph configurado (`public/og-image.jpg`)
- [ ] Sitemap ativo (`/sitemap.xml`)
- [ ] Robots.txt ativo (`/robots.txt`)
- [ ] `ads.txt` publicado (`/ads.txt`)

## Conteúdo e páginas

- [ ] Todas as páginas institucionais completas
- [ ] Política de Privacidade completa (`/privacidade`)
- [ ] Termos de Uso completos (`/termos`)
- [ ] Política de Cookies completa (`/cookies`)
- [ ] Página Sobre completa (`/sobre`)
- [ ] Página Quem Somos completa (`/quem-somos`)
- [ ] Página Contato funcionando (`/contato`)
- [ ] Menu funcionando sem links quebrados
- [ ] Rodapé completo com links institucionais
- [ ] Nenhuma página em construção
- [ ] Nenhum link quebrado
- [ ] Nenhum botão sem destino
- [ ] Conteúdo original no blog (mínimo 12 artigos úteis)
- [ ] Vagas com descrição útil
- [ ] Site responsivo no celular
- [ ] Sem páginas apenas com imagem
- [ ] Sem páginas sem texto
- [ ] Sem conteúdo proibido

## Anúncios

- [ ] `NEXT_PUBLIC_ADSENSE_CLIENT_ID` configurado somente após aprovação
- [ ] Sem anúncios excessivos antes da aprovação
- [ ] Sem pop-up de anúncio
- [ ] Sem redirecionamento automático para anúncios
- [ ] Espaços de anúncio discretos e fora do conteúdo principal

## Cookies e privacidade

- [ ] Banner de cookies funcionando
- [ ] Política de cookies menciona AdSense e Analytics
- [ ] Consentimento registrado no navegador do usuário
